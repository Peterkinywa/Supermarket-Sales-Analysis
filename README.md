# Supermarket-Sales-Analysis

## Project objective
The objective of this project is to analyze a 3-Yr supermarket transaction dataset and extract meaningful insights related to product sales, payment methods and customer behavior using spreadsheet-based data analysis tools.

## Project description
I was provided with a CSV file named **supermarket_transactions.csv**, containing transactional data collected from supermarkets across Australia over a period of three years. Acting as a Junior Data Analyst, I was required to clean, analyze and
summarize the data to answer specific business questions provided by our team lead.
